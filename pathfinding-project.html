Pathfinding-Visualizer

Description

This project is a web application that allows users to visualize some of the more well know pathfinding algorithms. The motivation for this project came from my interset in pathfinding algorithms combined with the fact that, at the time, I was also studying various pathfinding algorithms to prepare for my technical coding interviews.

Development

I decided to create this project in ReactJS because I wanted to make a web application and ReactJS is also a very versatile and dynamic programming language for desgining interactive websites. I had never created a project in ReactJS prior to this, so I had to watch multiple tutorials and read through pages of documentation to get a grasp on how I was going to build this project from scratch. Once I got a handle on ReactJS, the most challenging part was implementing the animation. Coding the actual pathfinding algorithms was fairly straightfoward because I had coded these algorithms in various other languages previously.

This entire project was coded in ReactJS however ReactJS inherently incorporates HTML, CSS, and Javascript in the actual syntax.
